## Description

A simple NestJs API to retrieve data from Strava API.

## Installation

```bash
$ npm install
```

## Configuration

You must provide the .env file with your Strava id et Strava token.

You can fetch your Strava token at this url : https://www.strava.com/settings/api

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
